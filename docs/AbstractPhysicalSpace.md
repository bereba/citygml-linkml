

# Class: AbstractPhysicalSpace 


_AbstractPhysicalSpace is the abstract superclass for all types of physical spaces. Physical space refers to spaces that are fully or partially bounded by physical objects._




* __NOTE__: this is an abstract class and should not be instantiated directly
